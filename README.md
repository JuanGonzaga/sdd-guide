# SDD Guide — Claude Code + Speckit

Landing page interativa para apresentar um fluxo de desenvolvimento com IA usando Claude Code, SDD, Speckit e skills customizadas.

## Estrutura

```text
.
├── index.html
├── README.md
└── resources/
    ├── legacy-context-pack.zip
    ├── speckit-workflow-pack.zip
    └── productivity-pack.zip
```

## Publicação

Suba `index.html`, `README.md` e a pasta `resources/` na raiz do repositório e ative GitHub Pages em `main` / root.
